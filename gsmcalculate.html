GSM Calculation

Find out GSM if you have fabric weight, length and width

Formula: GSM=(kg X 1550000) / (yards X 36 X width)


NB: Please note that in our calculator you need to give width in inch.

Find out Weight if you have gsm, length and width

Formula: KG=(gsm X yards X 36 X width) / 1550000


NB: 1 yards = 36 inch
