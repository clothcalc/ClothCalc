Fabric Conversion

Convert Fabric from Yards to Kg, If your fabric consumption in yard

Formula 1:
Garment consumption (in Kg.) = (Length X GSM X width in meter)/1000
Formula 2:
length(yard) X 36 X width(inch) X GSM / 1550000 (1 yard = 36 inches)

kg to yards, fabric width in inch

Formula:
Kg to Yards = (Fabric weight * 1000)/(Fabric GSM *Fabric Width in inch * 0.0254)*1.09361 (when fabric width is given in inches)

kg to yards, fabric width in centimeters

Formula:
Yards = (Weight in kg * 1000) / (GSM * Fabric Width in meters * 1.09361)
